HTML Notes

HTML Basics

HTML stands for HyperText Markup Language. It is the standard markup language used for creating web pages. Some HTML elements have no content. These elements are called empty elements. Empty elements do not have an end tag!HTML is a markup language that defines the structure of your content. HTML consists of a series of elements, which you use to enclose, or wrap, different parts of the content to make it appear a certain way, or act a certain way. The enclosing tags can make a word or image hyperlink to somewhere else, can italicize words, can make the font bigger or smaller, and so on.

HTML Tags

HTML documents are made up of HTML tags. Tags are used to mark up elements on a webpage. Some common tags include <p> for paragraphs, <a> for links, and <img> for images.There is a range of HTML tags, they help you to design your web page. There are four required tags in HTML. These are html, title, head and body. HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.

HTML Attributes

HTML attributes provide additional information about HTML elements. Attributes are specified within the opening tag of an element and consist of a name and a value. For example, the src attribute specifies the source URL of an image in the <img> tag.All HTML elements can have attributes Attributes provide additional information about elements Attributes are always specified in the start tag Attributes usually come in name/value pairs like: name="value" The href Attribute is defines a hyperlink. The href attribute specifies the URL of the page the link.The src Attribute is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed.

HTML Headings

HTML provides six levels of headings, from <h1> to <h6>. Headings are used to structure and organize the content on a webpage.HTML defines six levels of headings. A heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading. The heading elements are H1, H2, H3, H4, H5, and H6 with H1 being the highest (or most important) level and H6 the least.
